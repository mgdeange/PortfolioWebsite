EGR 314 Project: Mobile Plant Caretaker

Abstract

The team designed and built a mobile weather station that continously transmits data to a WiFi server for a user to read from, and allow the user to control from that server. I was in charge of the PCB design, code, and documentation for the project, which you can find here.

The Problem

The mission of the team is to create a sensor-based weather-reporting product that is unique, portable, professional, reliable, and user-friendly. It is something that should be focused so that a person can easily use and install themselves and is relatively simple to collect data from, along with contingency plans in case things go awry.

The Solution

When it comes to plant caretaking, some plants require more than just watering them every now and again Some require a more robust maintenance schedule, which can interefere with people's working schedules. The project seeks to offset that by creating a device that can autonomously care for a plant based on certain factors, such as humidity of the soil or the temperature of the surrounding area, and signal to the user how the plant is doing wherever they are.

Design Features

The mobile weather station consists of a belt-based movement system that the plant would be placed on, and move to and from a shaded area in accordance to certain variables, such as humidity and temperature.
